# 4-cylinder-piston-engine-Solidworks-
A complete CAD model of a four-cylinder aircraft piston engine developed in SolidWorks.  The project was created to strengthen my understanding of piston engine architecture, mechanical assemblies, and parametric CAD modeling while improving proficiency in SolidWorks.

## Features
- Complete four-cylinder engine assembly
- Individually modeled engine components
- Parametric part modeling
- Assembly constraints and mates
- Engineering drawings
- High-quality rendered images

## Major Components
-Piston
-Piston Ring
-Crankshaft
-Connecting rod
-Connecting rod Cap
-Piston Pin
